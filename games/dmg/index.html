DMG (2017)

A small SHUMP made for the Game Boy using GBDK.


Also, it works on real hardware :)

Playable on the web (without sound) thanks to JSGB.

- Controls -

Arrow keys - to move around.

A key - Start

S key - Select

Z key - A (Shoot)

C key - B
